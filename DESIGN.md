# 🎨 Визуальный дизайн карточек игроков

## Структура карточки

```
┌─────────────────┐
│ ●1        👑    │  ← Номер (слева) + Иконка роли (справа)
│                 │
│                 │
│     [ФОТО]      │  ← Фото игрока 120x160px (3:4)
│                 │
│                 │
│  ┌───────────┐  │
│  │  Игрок1   │  │  ← Ник с полупрозрачным фоном
│  └───────────┘  │
└─────────────────┘
```

## Компоненты карточки

- **Скругление углов**: 8px
- **Тень**: `box-shadow: 0 4px 8px rgba(0,0,0,0.3)`
- **Обработка**: `object-fit: cover` (обрезка по размеру)

- **Цвет текста**: Белый (#fff)
- **Шрифт**: 16px, жирный
- **Позиция**: `top: 8px; left: 8px`

Отображается только для Дона, Мафии и Шерифа.

- **Позиция**: `top: 8px; right: 8px`

## Состояния карточки

### Активный игрок (в игре)
```
┌─────────────────┐
│ ●1        🔫    │
│                 │
│     [ФОТО]      │  ← Полная яркость
│                 │
│  ┌───────────┐  │
│  │  Игрок2   │  │
│  └───────────┘  │
└─────────────────┘
```
- **Прозрачность**: 100% (opacity: 1)
- **Фильтр**: Нет
- **Позиция**: Нормальная (transform: translateY(0))

### Выбывший игрок (убит/заголосован)
```
┌─────────────────┐
│ ●5              │  ← Серое, полупрозрачное
│                 │
│   [СЕРОЕ ФОТО]  │  ← 40% прозрачность + grayscale
│                 │     Опущено на 20px вниз
│  ┌───────────┐  │
│  │  Игрок5   │  │
│  │   Убит    │  │  ← Статус красным
│  └───────────┘  │
└─────────────────┘
```
- **Прозрачность**: 40% (opacity: 0.4)
- **Фильтр**: `grayscale(100%)` (черно-белое)
- **Позиция**: Опущена (transform: translateY(20px))
- **Тень фото**: `0 4px 8px rgba(255,0,0,0.5)` (красная)
- **Статус**: Красный текст "Убит" или "Заголосован"

## Анимация перехода

### При выбывании игрока
```css
transition: all 0.5s ease;
```

**Что меняется:**
1. Прозрачность плавно снижается до 40%
2. Карточка плавно опускается на 20px вниз
3. Включается серый фильтр
4. Тень фото становится красной

**Длительность**: 0.5 секунды
**Функция**: ease (плавное ускорение и замедление)

## Адаптивность

### Для разных разрешений
- **1920x1080 (FullHD)**: Стандартный размер 120x160px
- **2560x1440 (2K)**: Можно увеличить до 160x213px
- **3840x2160 (4K)**: Можно увеличить до 240x320px

### Модификация размеров
Измените следующие значения в CSS:
```css
.player-photo-container {
  width: 120px;   /* Базовая ширина */
  height: 160px;  /* Базовая высота (width * 4/3) */
}

.player-card img {
  width: 120px;
  height: 160px;
}
```

**Важно**: Сохраняйте соотношение 3:4 для корректного отображения фотографий.
